What Chrome Extensions are

What Chrome Extensions are
  • Programs that enhance Google Chrome's functionality
  • Written in client side language specification(HTML, CSS, Javascript)
  • Assisted with buildin functionality given by chrome as APIs
  • Intalled easily, updated automatically.

Why you should work on extension

Why you should work on extension
  • Easy to build and use
  • Makes your day to day work faster and easier
  • It fun to build tools

Structure of Chrome Extension

    Structure of an Extension
  • Manifest File - manifest.json
  • Browser Action or Page Action
  • Content Script
  • Background Page
  • Context Menu
    Overview of the Extension API

    chrome is the top level object and exposes

  • chrome.extension.*
  • chrome.browserAction.*
  • chrome.pageAction.*
  • chrome.bookmarks.*
  • chrome.tabs.*
  • chrome.windows.*
    Other APIs

    Extension can also use

  • Standard JavaScript and DOM APIs
  • XMLHttpRequest (ajax)
  • HTML 5 APIs

content script

    Content Page
  • - No access is required to the websites source files
  • - The extension shares assets with the page, so matching styling is easy
  • - It allows one to visualise a demo or feature as if it were in production
  • - No persistance of data
  • - Runs in tabs
  • - Limited chrome.* APIs

Communication between components

Sending from content script->bg or popup, bg->popup

                        
chrome.runtime.sendMessage({greeting:"hello"},function(response){
    console.log(response.farewell);
});

Sending from background script to content script

                        
chrome.tabs.query({active:true,currentWindow:true},function(tab){
    chrome.tabs.sendMessage(tab[0].id, {greeting:'hello'},function(response){
        console.log(response.farewell);
    });
});

Receiving from all

                        
chrome.runtime.onMessage.addListener(function(request, sender, sendResponse) {
    console.log(sender.tab?"from content script" + sender.tab.url:"from the extension");
    if(request.greeting==='hello'){
        sendResponse({farewell:"goodbye"});
    }
});
